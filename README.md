# covoiturage-universitaire

Plateforme de covoiturage universitaire développée en Java dans le cadre du module AGL.

## Description
Ce projet a pour objectif de concevoir et développer une application de covoiturage universitaire permettant aux étudiants de rechercher, proposer et réserver des trajets.

## Technologies
- Java
- GitHub
- Scrum
- UML
- Figma

## Répartition des fonctionnalités
- Elaa : Product Owner + Authentification
- Ghaya : Scrum Master + Réservation
- Yasmin : Recherche
- Maissa : Conducteur
- Ines : Administration

## Livrables réalisés
- Diagramme de cas d’utilisation
- Diagramme de classes
- Product Backlog
- Kanban Sprint 1
- Maquettes des différentes fonctionnalités
- Captures des livrables dans le dossier `docs/captures`

## Sprint 1
Le Sprint 1 correspond à une première version simple et démontrable du projet.  
Il comprend :
- la modélisation UML
- l’organisation initiale du backlog
- la mise en place du Kanban
- les maquettes de base
- une première implémentation Java simple
- des scénarios de base

## Sprint 2
Le Sprint 2 correspond à l’enrichissement du même projet.  
Il comprend :
- l’amélioration du code Java
- l’ajout de contrôles de saisie
- des méthodes métier plus complètes
- des scénarios plus riches
- une meilleure intégration entre les fonctionnalités
- l’amélioration des maquettes et de la navigation
